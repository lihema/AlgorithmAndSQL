###3.设计模式
````
1.创建型
常用：
单例模式、工厂模式、建造者模式

不常用：
原型模式

2.结构型
常用：
代理模式、桥接模式、装饰着模式、适配器模式

不常用：
门面模式、组合模式、享元模式


3.行为型
常用的有：观察者模式、模板模式、策略模式、职责链模式、迭代器模式、状态模式

不常用的有：访问者模式、备忘录模式、命令模式、解释器模式、中介模式

````
